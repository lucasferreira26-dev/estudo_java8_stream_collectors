# 📚 Java 8 - Streams & Collectors

Este projeto contém uma série de exercícios práticos utilizando **Streams e Collectors** do Java 8, com o objetivo de reforçar conceitos fundamentais como processamento funcional, manipulação de coleções e escrita de código mais declarativo.

---

## 🚀 Tecnologias utilizadas

- Java 8+
- IntelliJ IDEA
- Programação Funcional (Streams API)

---

## 🧠 Conceitos abordados

- `stream()`
- `filter()`
- `map()` / `mapToDouble()`
- `distinct()`
- `forEach()`
- `max()`
- `Comparator`
- `collect()`
- `Collectors`
- `summarizingDouble()`
- `Optional`

---

## 📌 Exercícios implementados

### 🔹 Desafio 1
Filtrar pedidos pagos com valor acima de R$500, mapear os clientes e remover duplicatas.

---

### 🔹 Desafio 2
Filtrar pedidos pendentes e calcular a soma total dos valores.

---

### 🔹 Desafio 3
Encontrar o pedido mais caro utilizando `max()` com `Comparator`.

---

### 🔹 Desafio 4
Encontrar o pedido mais recente com base na data.

---

### 🔹 Desafio 5
Gerar estatísticas completas dos pedidos (mínimo, máximo, média, soma e quantidade), desconsiderando pedidos cancelados.

---

### 🔹 Desafio 6
Listar clientes com pedidos cancelados, removendo nomes duplicados.

---

## 💡 Aprendizados

Durante a implementação desses exercícios, foi possível:

- Entender o conceito de **pipeline de dados** com streams  
- Aplicar filtros e transformações de forma encadeada  
- Trabalhar com agregações e estatísticas  
- Melhorar a legibilidade do código utilizando abordagem declarativa  
- Compreender melhor a diferença entre operações intermediárias e terminais
